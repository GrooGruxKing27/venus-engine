# Project Venus – MVP Vision

## What This Is
Project Venus is an MVP experiment to determine whether an explainable
outfit compatibility engine can help people confidently match clothing
based on color and pattern.

## In Scope (MVP)
- Upload one or two clothing images
- Extract dominant colors
- Classify pattern at a basic level (solid vs patterned)
- Produce a compatibility score (0–100)
- Explain why the score was given

## Out of Scope (For Now)
- Mobile apps
- Wardrobe storage
- Social features
- Machine learning model training
- Monetization

## Success Criteria
The MVP is successful if:
- Two images can be scored
- The explanation feels reasonable to a human
- The result is repeatable and consistent
